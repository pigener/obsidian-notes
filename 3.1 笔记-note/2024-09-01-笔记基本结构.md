#### 快捷键

- `Cmd/Ctrl + w`: 关闭当前笔记页面
    - `Cmd/Ctrl + n`: 创建一个新笔记页面
    - `Cmd/Ctrl(长按) + 点击笔记`: 在新窗口打开笔记
- `Cmd/Ctrl + 数字 1️⃣/2️⃣`: 跳到第  [数字] 1️⃣/2️⃣ 个页面窗口
    - `Cmd/Ctrl + o`: 打开全局搜索笔记框
    - `Cmd/Ctrl + b`: 加粗/取消加粗文字
